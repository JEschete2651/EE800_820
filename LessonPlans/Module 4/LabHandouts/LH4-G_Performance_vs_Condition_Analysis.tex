\labheader{4-G}{Stratified Accuracy by SNR, Distance, and Active-Station Count}

\section{Objective}
Produce the three stratified-accuracy curves that describe how each classifier \emph{degrades} (or does not) as operating conditions worsen: accuracy vs.\ \mbox{SNR}, accuracy vs.\ distance, and accuracy vs.\ active-station count. These curves are the robustness evidence used in the LH4-H final model selection rubric and they reveal which classifier family scales best to realistic deployment conditions.

\section{Prerequisites}
\begin{itemize}[leftmargin=2em]
	\item Lab Handouts~4-C through~4-F complete (RF, \mbox{SVM}, \mbox{MLP}, \mbox{CNN} all trained per task; LH4-F's \texttt{src\_test.npy} alignment files present alongside \texttt{Xs\_test.npy}).
	\item \texttt{merged\_dataset.csv} present at \texttt{Code/Tools/CampaignCollect/merged\_dataset.csv}.
	\item Campaign metadata present in the merged CSV: \texttt{distance\_m} populated (currently 3 m everywhere because C4 was skipped --- the distance plot is auto-skipped if the column has $<$2 unique values), \texttt{run\_id} prefixed by \texttt{C5a}/\texttt{C5b} for the active-station-count stratifier.
\end{itemize}

\begin{tcolorbox}[infobox={Sub-campaign encoding in this dataset}]
\texttt{collect.py} stamps both C5a and C5b rows with \texttt{campaign\_id = "C5"}; the sub-config is encoded in \texttt{run\_id} (\texttt{C5a\_<timestamp>}, \texttt{C5b\_<timestamp>}). The script's active-station-count filter therefore matches on \texttt{run\_id.startswith(...)}, not on \texttt{campaign\_id}. The filters here are cosmetic slicers, not leakage guards --- the leakage guard is the row-level stratified split from LH3-H bis (degenerate run-level split fell back to row-level).
\end{tcolorbox}

\section{Background: Average Accuracy Lies}
Two models can have identical top-line test accuracy while behaving very differently in the operating regime that matters. Model~A might do very well in high-\mbox{SNR} conditions and poorly in low-\mbox{SNR}; model~B might be uniformly mediocre. If the deployment environment is low-\mbox{SNR}-dominated, B is the better choice even though A has the higher headline number. Stratified accuracy surfaces this.

\section{Procedure}

\subsection{Step 1 --- Logging, the row-level test set, and the model dispatcher}
\texttt{Code/AIML/stratified.py} follows the LH4-A--F logging pattern. The script rebuilds the per-row test slice from the merged CSV (using the same \texttt{inter\_arrival\_ms > 0} filter LH4-B applied) so the raw column values (\texttt{snr\_db}, \texttt{distance\_m}, \texttt{run\_id}) are available for stratifying. Models that consume per-row vectors (RF, SVM, MLP) get the same scaling LH4-B saved; the CNN consumes \texttt{Xs\_test.npy} and is aligned to the merged-CSV row index via the \texttt{src\_test.npy} files written by LH4-F.

\begin{lstlisting}[language=Python, caption={Per-task predictions aligned to merged-CSV row indices.}]
def predict_row_model(task, model_kind):
    """Return (y_pred, y_true, src_idx) for RF/SVM/MLP."""
    pre = joblib.load(ARTIFACTS_DIR / task / "preprocessor.pkl")
    feats = pre["feature_cols"]
    raw = test_df[LABEL_COL[task]].to_numpy()
    known = np.isin(raw, pre["encoder"].classes_)
    sub = test_df[known]
    if len(sub) == 0:
        return None, None, None
    X = pre["scaler"].transform(sub[feats].to_numpy(dtype=np.float32))
    y_true = pre["encoder"].transform(sub[LABEL_COL[task]])
    if model_kind in ("rf", "svm"):
        bundle = joblib.load(ARTIFACTS_DIR / task / f"{model_kind}.pkl")
        y_pred = bundle["model"].predict(X)
    elif model_kind == "mlp":
        from mlp_train import MLP
        pkl = torch.load(ARTIFACTS_DIR / task / "mlp.pt", weights_only=False)
        m = MLP(pkl["n_features"], pkl["n_classes"]).to(DEVICE)
        m.load_state_dict(pkl["state_dict"]); m.eval()
        with torch.no_grad():
            y_pred = m(torch.from_numpy(X).float().to(DEVICE)
                       ).argmax(1).cpu().numpy()
    else:
        raise ValueError(model_kind)
    return y_pred, y_true, sub.index.to_numpy()


def predict_cnn(task):
    from cnn_train import CNN1D
    pkl = torch.load(ARTIFACTS_DIR / task / "cnn.pt", weights_only=False)
    Xs = np.load(ARTIFACTS_DIR / task / "Xs_test.npy")
    ys = np.load(ARTIFACTS_DIR / task / "ys_test.npy")
    src = np.load(ARTIFACTS_DIR / task / "src_test.npy")
    if len(Xs) == 0:
        return None, None, None
    m = CNN1D(pkl["n_features"], pkl["n_classes"]).to(DEVICE)
    m.load_state_dict(pkl["state_dict"]); m.eval()
    with torch.no_grad():
        y_pred = m(torch.from_numpy(Xs).float().to(DEVICE)
                   ).argmax(1).cpu().numpy()
    return y_pred, ys, src


def predict(task, model_kind):
    if model_kind == "cnn":
        return predict_cnn(task)
    return predict_row_model(task, model_kind)
\end{lstlisting}

\subsection{Step 2 --- Stratified accuracy with Wilson CI}
The script computes per-bucket accuracy with a Wilson 95\% CI for every model. Buckets with $<$30 samples are flagged in the log (the variance there is high enough that small between-model differences should not be over-read).

\begin{lstlisting}[language=Python]
def wilson_ci(n_correct, n_total, alpha=0.05):
    if n_total == 0:
        return 0.0, 1.0
    p = n_correct / n_total
    z = norm.ppf(1 - alpha / 2)
    denom = 1 + z * z / n_total
    center = (p + z * z / (2 * n_total)) / denom
    half = z * sqrt(p * (1 - p) / n_total
                    + z * z / (4 * n_total * n_total)) / denom
    return max(0.0, center - half), min(1.0, center + half)


def stratified_acc(task, stratifier_col, edges_or_values, ax, title):
    handles = []; bucket_summary = {}
    for model_kind in ["rf", "svm", "mlp", "cnn"]:
        y_pred, y_true, src = predict(task, model_kind)
        if y_pred is None:
            continue
        strat = df_full.loc[src, stratifier_col].to_numpy()
        accs = []; ns = []; los = []; his = []; centers = []
        is_edges = (isinstance(edges_or_values, np.ndarray)
                    and edges_or_values.dtype.kind in "fc"
                    and len(edges_or_values) > 1
                    and not np.array_equal(edges_or_values, np.unique(edges_or_values)[:1]))
        if is_edges and len(edges_or_values) > 2:
            interior = edges_or_values[1:-1]
            idx = np.digitize(strat, interior)
            n_b = len(edges_or_values) - 1
            for b in range(n_b):
                m = (idx == b); n = int(m.sum())
                center = 0.5 * (edges_or_values[b] + edges_or_values[b+1])
                centers.append(float(center))
                if n == 0:
                    accs.append(np.nan); ns.append(0); los.append(np.nan); his.append(np.nan)
                    continue
                nc = int((y_pred[m] == y_true[m]).sum())
                accs.append(nc / n); ns.append(n)
                lo, hi = wilson_ci(nc, n); los.append(lo); his.append(hi)
        else:
            for v in edges_or_values:
                m = (strat == v); n = int(m.sum())
                centers.append(float(v))
                if n == 0:
                    accs.append(np.nan); ns.append(0); los.append(np.nan); his.append(np.nan)
                    continue
                nc = int((y_pred[m] == y_true[m]).sum())
                accs.append(nc / n); ns.append(n)
                lo, hi = wilson_ci(nc, n); los.append(lo); his.append(hi)
        for c, n in zip(centers, ns):
            if 0 < n < 30:
                log.warning(f"  {task} {model_kind} bucket center={c:.2f}: "
                            f"n={n} (<30, noisy)")
        h, = ax.plot(centers, accs, "-o", label=model_kind.upper())
        handles.append(h)
        bucket_summary[model_kind] = {
            "centers": centers, "accs": accs, "ns": ns,
            "ci_lo": los, "ci_hi": his,
        }
    ax.set_title(title); ax.grid(True); ax.legend(handles=handles, fontsize=7)
    return bucket_summary
\end{lstlisting}

\subsection{Step 3 --- Per-task three-panel figure (SNR / distance / stations)}
The distance panel is auto-skipped if \texttt{distance\_m} has $<$2 unique values (the case in the current dataset because C4 was skipped). The stations panel is auto-skipped if the test split contains only one of \texttt{C5a}/\texttt{C5b}.

\begin{lstlisting}[language=Python]
def plot_task(task):
    if task == "mod":
        log.info(f"  {task}: only LoRa collected --- skipping")
        return None
    fig, axes = plt.subplots(1, 3, figsize=(15, 4))

    # Panel 1: SNR quintiles
    snrs = test_df["snr_db"].to_numpy()
    edges = np.quantile(snrs, np.linspace(0, 1, 6))
    snr_summary = stratified_acc(task, "snr_db", edges, axes[0],
                                 f"{task}: acc vs SNR (quintiles)")
    axes[0].set_xlabel("SNR (dB)"); axes[0].set_ylabel("Test accuracy")

    # Panel 2: distance (auto-skip if no variation)
    distances = sorted(test_df["distance_m"].unique())
    if len(distances) < 2:
        axes[1].set_title(f"{task}: distance plot SKIPPED")
        axes[1].text(0.5, 0.5, "C4 not collected\n(distance_m constant)",
                     ha="center", va="center", transform=axes[1].transAxes)
        dist_summary = None
    else:
        dist_summary = stratified_acc(task, "distance_m",
                                      np.asarray(distances), axes[1],
                                      f"{task}: acc vs distance")
        axes[1].set_xlabel("Distance (m)")

    # Panel 3: active stations (C5a -> 1, C5b -> 2)
    runs = test_df["run_id"].to_numpy().astype(str)
    stations = np.where(np.char.startswith(runs, "C5b"), 2,
                np.where(np.char.startswith(runs, "C5a"), 1, 0))
    df_full.loc[test_df.index, "_stations"] = stations
    discrete = sorted(set(stations.tolist()) - {0})
    if len(discrete) < 2:
        axes[2].set_title(f"{task}: stations plot SKIPPED")
        axes[2].text(0.5, 0.5, "Need both C5a and C5b in test",
                     ha="center", va="center", transform=axes[2].transAxes)
        sta_summary = None
    else:
        sta_summary = stratified_acc(task, "_stations",
                                     np.asarray(discrete), axes[2],
                                     f"{task}: acc vs active stations")
        axes[2].set_xlabel("Active stations"); axes[2].set_xticks(discrete)

    fig.tight_layout()
    out = FIGS_DIR / f"stratified_{task}.png"
    fig.savefig(out, dpi=150); plt.close(fig)
    log.info(f"  Saved {out}")
    return {"snr": snr_summary, "distance": dist_summary, "stations": sta_summary}


task_results = {}
for task in ["sf", "mod", "beacon", "pkt"]:
    log.info(f"--- Task: {task} ---")
    task_results[task] = plot_task(task)
\end{lstlisting}

\subsection{Step 4 --- Worst-bucket robustness score}
Defensible single-number robustness metric: the \emph{minimum} accuracy across all SNR buckets. A model with high average accuracy but terrible worst-case accuracy is a worse deployment candidate than one with slightly lower average and higher worst-case.

\begin{lstlisting}[language=Python]
log.info("=== Worst-bucket robustness (SNR quintiles) ===")
log.info(f"{'Task':<8} {'Model':<5} {'min':>8} {'mean':>8} {'max':>8}")
robustness = {}
for task, res in task_results.items():
    if not res or not res.get("snr"):
        continue
    robustness[task] = {}
    for model_kind, info in res["snr"].items():
        accs = np.asarray(info["accs"], dtype=float)
        mn   = float(np.nanmin(accs))
        mx   = float(np.nanmax(accs))
        mean = float(np.nanmean(accs))
        log.info(f"{task:<8} {model_kind:<5} {mn:>8.3f} {mean:>8.3f} {mx:>8.3f}")
        robustness[task][model_kind] = {"min": mn, "mean": mean, "max": mx}

manifest = {
    "generated_at": datetime.now(timezone.utc).isoformat(),
    "run_stamp":    run_stamp,
    "log_archive":  str(log_archive),
    "results":      task_results,
    "robustness":   robustness,
}
with open(MANIFEST_PATH, "w", encoding="utf-8") as f:
    json.dump(manifest, f, indent=2, default=str)
log.info(f"Wrote {MANIFEST_PATH}")
log.info("=== stratified.py done ===")
\end{lstlisting}

\subsection{Step 5 --- Narrative discussion (required for the deliverable)}
Write 2--3 paragraphs addressing:

\begin{enumerate}[leftmargin=2em]
	\item Which model degrades most gracefully as \mbox{SNR} drops on the SF task? Is this the same model that has the highest average accuracy in LH4-C/D/E/F?
	\item Does the \mbox{CNN}'s temporal input give it an advantage in the multi-station regime (C5b), where DATA and ACK packets from two different stations interleave? Compare per-bucket accuracy on the active-station-count panel.
	\item Are any model families noticeably steeper (degrade faster) at the low-\mbox{SNR} end? What does that suggest about how each would perform in a low-\mbox{SNR} field deployment? If C4 has not been collected, state that the distance question cannot be answered from the current dataset.
\end{enumerate}

\begin{tcolorbox}[warning={Watch for Small-Bucket Artefacts}]
	The script logs a \texttt{[WARN]} for any bucket with $<$30 samples and stores the Wilson 95\% CI bounds for every bucket in \texttt{stratified\_manifest.json}. A bucket of $n=20$ at acc=0.90 has a CI of roughly $[0.70, 0.97]$; do not over-read between-model differences within that range. Quintile bucketing of a 327-row test set gives $\sim$65 rows per bucket on average, but tail buckets after stratification can be smaller.
\end{tcolorbox}
